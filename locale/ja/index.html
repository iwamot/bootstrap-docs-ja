Bootstrap

世界でいちばん人気のフロントエンドコンポーネントライブラリを使って、レスポンシブでモバイルファーストなプロジェクトをWeb上に構築しましょう。

Bootstrap は HTML、CSS および JS で開発するためのオープンソースのツールキットです。 Sass の変数やミックスイン、レスポンシブなグリッドシステム、豊富な組み込みコンポーネント、jQuery を利用したパワフルなプラグインを使って、アイデアのプロトタイプを手早く作ったり、アプリケーション全体を構築したりできます。

インストール

Bootstrap のソースである Sass や JavaScript ファイルを、npm や Composer、Meteor から取得してください。 パッケージ管理されたインストールにはドキュメントが含まれていませんが、ビルドシステムと readme が含まれています。

{% highlight sh %} npm install bootstrap {% endhighlight %} {% highlight sh %} gem install bootstrap -v {{ site.current_ruby_version }} {% endhighlight %}
